Forward Proxy Settings

Configure which proxy server the extension routes Chrome through.

Go proxy default port is 8081 (Node is 8080). If the proxy runs on another machine, use that server's IP or hostname — not localhost unless Chrome runs on the same host.

Hostname only — no http://, no port, no username/password.

407 test (from the same machine as Chrome):
curl -v -x http://HOST:PORT https://example.com -I
Do not use -U. You should see HTTP/1.1 407 Proxy Authentication Required. If curl cannot connect, fix host/port/firewall first. If curl gets 403 instead of 407, your client IP may be blocked in allowedClientIps on the server.
