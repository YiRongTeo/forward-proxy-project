HTTPS uses CONNECT. Chrome cannot inject headers on CONNECT — the extension answers the proxy 407 with your session ID (same as curl -U 'session1234:session'). Open Options and set port 8081 for Go (8080 for Node). Reload the extension after changes. After browsing, the service worker console should show onAuthRequired and authSupplied. If you only see refreshed, check Control in the popup — it must be controlled_by_this_extension.
